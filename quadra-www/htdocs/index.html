What is Quadra?

An addictive action puzzle game with single player and multiplayer capabilities (Internet or LAN).

We are looking for help!

Join the community of Quadra hackers and help out a great game! Even if you do not know much programming, you can still help in many ways:

  • do testing and report bugs
  • verify existing bugs
  • try to fix these bugs!
  • make binary packages
  • write documentation
  • work on the web site
  • help people out with Quadra
  • organize a tournament
  • run a dedicated server
  • show Quadra to other people
  • find other ideas like these!

If you want to join the community, do not hesitate! Simply join the appropriate(s) mailing lists and download a development release of Quadra (or better yet, use CVS to be at the bleeding edge)!

Resources

These are resources provided by SourceForge. They are also available from our project page.

  • Technical Support

    To place a technical support request, follow this link. Keep in mind that this project is based upon volunteer work, so if your request takes a while to be answered, be patient, we will do our best to answer as fast as we can!

  • Bug Tracking

    If you discover any bug or would like to write down an idea for an improvement to Quadra, go there and submit a bug so that we don't forget about it! If you have an account on SourceForge, make sure you are logged in so that you get notification of any changes related to your bug.

  • Patch Manager

    If you actually went ahead and fixed a bug or made a cool modification to Quadra, no matter how insignificant, please submit it to us for inclusion in the official source code.

  • Mailing Lists

    If you want to discuss Quadra with other users and hackers, get notifications of changes to the CVS repository or browse the mailing lists archives, this is the place to go.

  • CVS repository

    This is the home of the official source code. You can also obtain the source code using CVS.
